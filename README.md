
# 👗 Glotto Clothing Website

### Overview
Glotto Clothing is a **Botswana-based fashion brand** that blends cultural heritage with modern design.  
This responsive website showcases the brand’s collections, sustainability efforts, and values, while connecting with customers locally and globally.  

---

### 🧭 Project Structure
- `index.html` → Main webpage structure  
- `style.css` → Custom styles and responsive layout  
- `images/banner.jpg` → Hero banner image  
- `images/look1.jpg`, `images/look2.jpg`, `images/look3.jpg` → Lookbook visuals  

---

### ✨ Features
- Responsive Navbar with smooth scrolling  
- Hero Section with brand tagline and call-to-action button  
- About Section detailing Glotto’s story and mission  
- Collections Section featuring Men’s, Women’s, and Accessories categories  
- Lookbook Gallery showcasing styled outfits  
- Sustainability Section highlighting eco-friendly practices  
- Contact Section with email and social media links  
- Footer with copyright and brand statement  

---

### 💻 Technologies Used
- **HTML5** – Semantic structure  
- **CSS3** – Custom styling and responsive design  
- **Bootstrap (optional)** – For layout and grid system  
- **JavaScript (Bootstrap Bundle)** – For interactivity  

### 🧵 Brand Identity
- **Name:** Glotto Clothing  
- **Origin:** Botswana  
- **Tagline:** *Wear Your Heritage*  
- **Values:** Authenticity, Sustainability, Community Empowerment
